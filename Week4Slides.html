Purpose

  • Create a simple R Shiny application
    • Nothing extensive, just to demonstrate some Shiny functionality
  • Deploy application on Shiny web site
    • Demonstrate the use of Shiny tools
    • Showcase the publishing of the application for widespread distributed use
  • Create a Slidify slideshow to accompany the application

Application Overview

  • Provides a simple view of the "women" data set found in R
  • Displays a chart showing the relationship between the height and weight of women and a predicted value of weight provided a height

User Interface

  • Functions
    • To show the smooth line in the chart, simply check the box to show, or un-check to not show
    • To show a predicted weight value, fill in a height (in inches) in the text box and the weight value will populate a moment later without any button presses

Regression Modeling

  • The purpose of this exercise is not to create an elaborate regression model
  • Simply chose a linear model with the formula:
model <- lm(weights ~ heights, data = heightsAndWeights)
  • Linear model produced the following coefficients:
## (Intercept)     heights 
##   -87.51667     3.45000
  • Equation to predict weight is then:
predictedWeight <- height*model$coefficients[2] + model$coefficients[1]

Notes

  • There is no input validation performed.
    • There are values for height that will display negative weight values.
    • No error occurs if providing a negative height value
    • There is no upper-bounds on height either.
